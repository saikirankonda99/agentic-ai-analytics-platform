# Collaboration Screenshots

Use this folder for curated screenshots of shared workspace behavior.

Recommended captures:

- `01-shared-workspace-history.png`: shared team workspace selected in the sidebar with `History` showing recent collaboration activity.
- `02-shared-report-owner-metadata.png`: saved report or investigation with visibility, owner, creator, and updated-at metadata.
- `03-workspace-scope-switcher.png`: sidebar workspace scope control showing personal and shared team options.

Keep examples synthetic and avoid screenshots containing private names, credentials, or customer data.
